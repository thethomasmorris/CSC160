The HTML DOM is a way to get, change, add or delete HTML elements using JavaScript

All HTML elements are defined as objects

A property is a value you can get or set like changing the content of an HTML element.

A method (like a function) is an action you can do like adding or deleting an HTML element.

With the DOM, JavaScript has a way to create dynamic HTML:

  • Add new HTML elements
  • Change HTML elements
  • Change CSS on the page
  • Remove HTML elements
  • React to HTML events like a form button being clicked
  • Create new HTML events
